#######################################################################
Данный файл содержит описание проекта product-finder - 
веб-приложение для поиска товаров в прайс-листе. 

#######################################################################
Для сборки приложения необходимо:
	JDK 1.7
	maven 3

Сборка приложения:
	mvn clean pakcage

Запуск приложения:
	mvn tomcat:run

Приложнеие будет доступно по аресу:
	http://localhost:8080/product-finder/

#########################################################################
Для запуска приложения необходимо предворительно настроить подключение 
к СУБД и создать структуру базы данных.

В качестве СУБД используется сервер PostgresSQL версии 9.2.
Параметры подключения к БД хранятся в файле ресурсов:
	/product-finder/src/main/resources/META-INF/database.properties

Для тестирования используется втраиваемая СУБД HSQL версии 2.3.3.
Параметры подключения при тестировании:
	/product-finder/src/test/resources/META-INF/database.properties

Основные параметры необходимые для подключения это:
	
	#JDBC url подключения к БД
	dataSource.url=jdbc:postgresql://localhost:5432/product_finder_db
	#Имя пользователя БД
	dataSource.username=postgres
	#Пароль пользователя БД
	dataSource.password=111111

Иструкции по созданию базы данных содержаться в файле:
	/product-finder/src/main/resources/META-INF/sql/create.sql
Если необходимо предворительно инициализировать таблицы БД тестовыми данными, то необходимо выполенить скрипт:
	/product-finder/src/main/resources/META-INF/sql/data.sql
